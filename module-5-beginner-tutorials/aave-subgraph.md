# Aave Subgraph

Lend is a simple React component, leveraging the Aave subgraph for market data, a custom useAaveData react hook to fetch user data, and aave-js to prepare transactions. [Read more here](https://medium.com/@azfuller20/lend-with-aave-v2-draft-20bacceedade)!

The Aave Ape is a helper contract that lets you go long ("ape") the Aave asset of your choice, borrowing a "short" asset on Aave to buy more of the "ape" asset. It also lets you unwind your position with an Aave flashloan. Download & build this repository, then explore the source code to learn more.
